# Voice-Based-Parkinson-s-Disease-Detection

Project Overview

Voice-Based Parkinson's Disease Detection is a Machine Learning project that predicts whether a person has Parkinson's Disease using voice measurements. The model is trained using the UCI Parkinson's Dataset and applies data preprocessing, feature selection, model training, and performance evaluation to classify patients accurately.

Problem Statement

Parkinson's Disease is a neurological disorder that affects speech and movement. Early detection can improve treatment and patient care. This project aims to assist in identifying Parkinson's Disease from voice recordings using Machine Learning.

Dataset

Dataset: UCI Machine Learning Repository

Number of Samples: 195

Number of Features: 22 Voice Features

Target Variable:

0 → Healthy
1 → Parkinson's Disease

Dataset Link:

https://archive.ics.uci.edu/ml/datasets/parkinsons
